# Code of Conduct

## Our pledge

We want this project to remain respectful, inclusive, and helpful for everyone.

## Expected behavior

- Be respectful and constructive.
- Be patient with beginners.
- Focus on the problem, not the person.
- Assume good faith.

## Unacceptable behavior

- Harassment or insults of any kind.
- Discrimination based on identity.
- Deliberate disruption of discussions.

## Scope

This code of conduct applies to all project spaces, including issues and pull requests.
