# Support

## Where to get help

If you need help with this repository, start by checking:

- `README.md`
- `README.fr.md`
- `CONTRIBUTING.md`
- `SECURITY.md`

## Support channels

- **GitHub Issues** — for bugs or feature requests.
- **GitHub Discussions** — for general questions (enable in repository settings).
- **Pull requests** — for direct improvements.

## Before asking for help

Please include:

- What you were trying to do.
- What happened.
- What you expected to happen.
- Your environment (OS, Python version, etc.) if relevant.
